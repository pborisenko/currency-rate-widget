CRWidget Demo

Currency Rate Widget предоставляет данные курсов валют Центрального Банка России.

Опции по умолчанию

Вариант опций 1

     'favorite': ['USD', 'EUR', 'GBP'],
     'fullValue': true,
     'deltaPic': false
}

}
